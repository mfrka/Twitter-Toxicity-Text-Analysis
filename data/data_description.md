# Deskripsi Data

## Sumber Data
Data yang digunakan dalam penelitian ini diperoleh dari platform **Kaggle**.
Dataset berupa kumpulan teks Twitter yang telah diberi label berdasarkan tingkat toksisitas.

## Tujuan Penggunaan Data
Data ini digunakan untuk keperluan:
- Pembelajaran
- Analisis data teks
- Penyusunan portofolio pribadi

## Gambaran Umum Data
Dataset berisi teks tweet yang diklasifikasikan ke dalam dua kategori, yaitu:
- Toxic
- Non-Toxic

Setiap observasi merepresentasikan satu tweet.

## Struktur Variabel
Secara umum, dataset memiliki struktur sebagai berikut:
- `text` : isi teks tweet
- `label` : kategori toksisitas tweet (toxic / non-toxic)

## Catatan
Dataset tidak disertakan secara langsung dalam repository ini karena
pertimbangan lisensi dan ukuran file. Dataset dapat diunduh langsung
melalui platform Kaggle.
